# 🎬 Cineflix - Movie Ticket Booking System

A responsive movie ticket booking website with seat selection and payment flow.

### ✨ Features
- Movie listing with posters & showtimes
- Interactive seat selection layout
- Ticket booking & confirmation page
- Fully responsive design for mobile/desktop

### 🛠️ Tech Stack
HTML5, CSS3, JavaScript
